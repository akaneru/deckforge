
# deckforge

A cli utility to parse KeyForge decks and print some stats.

## usage

`$ dart lib/deckforge.dart <deck-id> <deck-id>...`

## example

`$ dart lib/deckforge.dart f2572901-8fd9-46f5-917d-48f409d8ec16 6cffebc8-8922-49ad-b26b-3b076830e7ee`
